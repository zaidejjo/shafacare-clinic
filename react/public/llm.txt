# Shafa Care Clinic 

## About
Shafa Care Clinic is a modern, family-focused medical clinic based in **Amman, Jordan**. Founded and led by **Dr. Firas Ajo**, the clinic delivers compassionate, patient-centered healthcare with a full spectrum of services under one roof.

## Technology Stack
- Built with React 19, TanStack Router, TanStack Start, Vite, and Tailwind CSS v4.
- Hosted on Cloudflare Pages at https://shafacare-clinic.pages.dev.

## SEO & Structured Data
- JSON-LD MedicalBusiness + LocalBusiness schema on the homepage with aggregate rating (4.9★, 30k+ reviews), all 8 services, and 4 practitioner profiles.
- Static XML sitemap at `/sitemap.xml` covering all 12 pages.
- Dynamic server-rendered sitemap at the same path.
- `robots.txt` allows all crawlers and points to the sitemap.
- hreflang tags for English and Arabic on every page.
- Open Graph and Twitter Card meta tags with og:image on all pages.
- Canonical URLs on every page to prevent duplicate content.
